闭包是指内部函数传递到所在作用域外部，它会持有对该作用域的引用。

#####   所有的回调函数都是闭包。
```
function wait(message){
    setTimeout(function timer(){   
        console.log(message);   // 持有wait函数作用域
    }, 1000)
}
```

示例：
```
for(var i = 1; i<=5; i++){
    setTimeout(function timer(){
        console.log(i)
        }, i*1000)
}
// 6个6 -> 延迟函数的回调会在循环结束时才执行
```

#####    解决方案
```
// 方案1
for(var i = 1; i<=5; i++){
    (function(i){
        setTimeout(function timer(){
            console.log(i)
            }, i*1000)
    })(i)
}
// 方案2
for(var i = 1; i<=5; i++){
    let j = i
    setTimeout(function timer(){
        console.log(j)
        }, j*1000)
}
```

#####   模块模式
```
var foo = (function CoolModule(){
    var something = 'cool';
    function doSomething(){
        console.log(something)
    }
    return {
        something
    }
})
foo.something()
```
